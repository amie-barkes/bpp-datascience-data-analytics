# bpp-datascience-data-analytics
## Synapse Analytics Notebooks Developed for the BPP Data Scientist Data Analytics Module
The objective of this project was to provide an introduction to performing Machine Learning techniques to solve a workplace problem. 
Claranet is a B2B suppier and as such Customer Churn is a real issue. The project sought to establish isf Customer Churn defined as the cancellation of Orders could be predicted based upon data avaialble in the uk-data-platfrom including Customer Incidents, Customer Relationship Survey Results and Customer Complaints.
